MAPS

In Maps, we have Four different types,
1. HashMap
2. LinkedHashMap
3. TreeMap
4. HashTable

Methods used in Maps

1. put():used to add data to Map object
2. get(key):used to get value based on key
3. remove(key):used to remove entry based on key
4. containsKey():used to verify whether key exist or not
5. containsValue():used to verify whether value exist or not
5. keySet():used to get only keys
7. values():used to get only values
8. entrySet():used to get both keys & values

Hash Map

--> HashMap provides implementation to Map interface
--> HashMap allow one null key & multiple null values
--> HashMap doesn't maintain any order
--> HashMap is not synchronized
--> HashMap support homogeneous as well as heterogeneous
--> HashMap object with homogeneous
--> HashMap maintain initial capacity of 16


LinkedHash Map

--> LinkedHashMap provides implementation to Map interface
--> LinkedHashMap allow one null key & multiple null values
--> LinkedHashMap maintain insertion order
--> LinkedHashMap is not synchronized
--> LinkedHashMap support homogeneous as well as heterogeneous
--> LinkedHashMap object with homogeneous
--> LinkedHashMap doesn't maintain initial capacity.

Tree Map

--> TreeMap provides implementation to Map interface
--> TreeMap doesn't null key,but allow multiple null values
--> TreeMap maintain data in ascending order
--> TreeMap support only homogeneous, because internally it use comparable interface compareTO() to sort data.
--> TreeMap maintain initial capacity of 16

Hash Table

--> HashTable provides implementation to Map interface
--> HashTable doesn't allow null key & null values
--> HashTable support homogeneous as well as heterogeneous
--> HashTable is synchronized
